ОБЩАЯ СТРУКТУРА КАЖДОГО ИЗ 2 СЛАЙДОВ:
Заголовок (фирменный зелёный) + строка-пояснение: «В карточке: веса 
(Корп/Функ/Инд) и итоговый % выплаты по оценке PR (1–5). Под примером — 
как считается.»
Ниже — ДВЕ карточки примеров В РЯД. Каждая карточка сверху вниз:
1) Тег «Пример N» + категория; строкой ниже — описание (кому).
2) Горизонтальная СЛОЖЕННАЯ ПОЛОСА весов Корп/Функ/Инд: ширина сегмента ∝ %, 
   внутри подпись («Корп 50%» и т.д.). Корп — светлый зелёный, Функ — средний 
   зелёный, Инд — тёмный/насыщенный зелёный. Если Функ = 0 — сегмента нет.
3) РЯД из 5 столбцов (оценки PR 1–5), в каждом сверху вниз:
   – бейдж-кружок с номером: 1 — #E0392F, 2 — #EE8A1E, 3 — #8C9196, 
     4 — фирменный зелёный, 5 — фирменный тёмно-зелёный; цифра белая;
   – KPR: −90% / −50% / 0% / +10% / +20%;
   – % выплаты в светлой плашке В ТОН оценки (светло-красная → светло-зелёная).
4) Тонкий разделитель, ниже — РАСЧЁТ ИМЕННО ЭТОГО примера, заголовок 
   «Как считается · оценка 4 (KPR +10%), выполнение 100%», строки:
     Корп: X% × 100% = X%
     Функ: Y% × 100% = Y%   (если Функ 0 — просто «0%»)
     Инд:  Z% × 100% × (1 + 10%) = …   ← «(1 + 10%)» выделить зелёным
   и итоговая светло-зелёная плашка: «Итого % выплаты = … = …%».

━━━ СЛАЙД 1 — «Как оценка PR влияет на выплату · примеры 1–2» ━━━
Пример 1 — БЕ Технологии · категория 1.
  Описание: CEO-3, CEO-4 (целевой ГБ > 20%), кроме кат. 2; CEO-5 и ниже 
  (целевой ГБ > 20%). Веса: Корп 50 / Функ 20 / Инд 30.
  % выплаты по оценкам 1–5: 73 / 85 / 100 / 103 / 106.
  Расчёт (оценка 4): Корп 50%×100%=50%; Функ 20%×100%=20%; 
  Инд 30%×100%×(1+10%)=33%; Итого = 50 + 20 + 33 = 103%.
Пример 2 — БЕ Технологии кат. 2 · БЕ Роботикс.
  Описание: Дирекции разработки/технологического развития; Роботикс — 
  CEO-4 и ниже. Веса: Корп 30 / Функ 20 / Инд 50.
  % выплаты: 55 / 75 / 100 / 105 / 110.
  Расчёт (оценка 4): Корп 30%×100%=30%; Функ 20%×100%=20%; 
  Инд 50%×100%×(1+10%)=55%; Итого = 30 + 20 + 55 = 105%.
Сноска слайда: «Оценка 3 — базовые 100%. При одной и той же оценке 4 выплата 
тем выше, чем больше вес индивидуальных КПЭ.»

━━━ СЛАЙД 2 — «Как оценка PR влияет на выплату · примеры 3–4» ━━━
Пример 3 — БЕ Технологии · категория 3.
  Описание: CEO-3 и ниже (целевой ГБ ≤ 20%). Веса: Корп 25 / Функ 0 / Инд 75.
